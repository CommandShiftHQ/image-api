Manchester Codes Image Sharing API

Endpoints

Users can authenticate themselves using either an email/password or with

POST /auth/login

Request Params:

  • email - users email address
  • password - users password

Response:

  • token - JWT containing the users details

POST /auth/google

Request Params:

  • email - users email address
  • password - users password

Response:

  • token - JWT containing the users details
